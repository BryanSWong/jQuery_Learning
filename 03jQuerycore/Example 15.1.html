Selecting Form Elements

jQuery offers several pseudo-selectors that help find elements in
forms. These are especially helpful because it can be difficult to
distinguish between form elements based on their state or type using
standard CSS selectors.

:button


Using the :button pseudo-selector targets any
< button> elements and elements with a type="button":

$("form :button");

:checkbox
Using the :checkbox pseudo-selector targets any
< input> elements with a type="checkbox":

$("form :checkbox");

:checked

Not to be confused with :checkbox, :checked targets checked
checkboxes, but keep in mind that this selector works also for
checked radio buttons, and select elements (for select elements
only, use the :selected selector):

$("form :checked");

:disabled

Using the :disabled pseudo-selector targets any
< input> elements with the disabled attribute:

$("form :disabled");

:enabled

Basically the inverse of the :disabled pseudo-selector,
the :enabled pseudo-selector targets any elements that do not
have a disabled attribute:

$("form :enabled");

:file

Using the :file pseudo-selector targets any
< input> elements that have a type="file":

$("form :file");

:image

Using the :image pseudo-selector targets any
< input> elements that have a type="image":

$("form :image");

:input

Using the :input selector selects all < input>, < textarea>,
< select>, and < button> elements:

$("form :input");

:password

Using the :password pseudo-selector targets any
< input> elements with a type="password":

$("form :password");

radio

Using the :radio pseudo-selector targets any
< input> elements that have a type="radio":

$("form :radio");

To select a set of associated radio buttons use:

$("form input[name='gender'] :radio");

:reset

Using the :reset pseudo-selector targets any < input>
elements that have a type="reset":

$("form :reset");

:selected

Using the :selected pseudo-selector targets any selected items in < option> elements:

$("form :selected");

:submit Using the :submit pseudo-selector targets any < button> or < input>
elements with a type="submit":

$("form :submit");

:text

Using the :text pseudo-selector targets any < input> elements with a type="text":

$("form :text");
