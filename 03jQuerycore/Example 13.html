The .attr() method

The .attr() method acts as both a getter and a setter. As a setter, .attr() can accept either a key and a value, or an object containing one or more key/value pairs.

.attr() as a setter:

code:

$("a").attr("href", "allMyHrefsAreTheSameNow.html");

$("a").attr({
title: "all titles are the same too!",
href: "somethingNew.html"
});
the orginal a link below is:
< a href="Example 01.html">< /a>

This link has been changed.

if you inspect the webpage you will see that the href has been changed.
also a title was added, the reason why you don't see "allMyHrefsAreTheSameNow.html"
is becuase the next $("a") call changed it to "somethingNew.html"
