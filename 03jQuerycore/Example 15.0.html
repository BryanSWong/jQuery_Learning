The most basic concept of jQuery is to "select some elements and do
something with them." jQuery supports most CSS3 selectors, as well
as some non-standard selectors.

Examples:

This is a paragraph with out the id property but the paragraph with
the id property is hidden below because it was selected and set to hide
with jquery. you can inspect the webpage and see the paragraph.

Here is is the code:
$("#myId).hide();

This is a paragraph with the id property = myId.


This is a paragraph with out a class id. The other paragraph has the class
id and is hidden due to jquery .hide() called on the selector of the class.
you can inspect the webpage and see paragraph.

Here is is the code:
$(".myClass").hide();

This paragraph has the class id = myClass.

This form does not have the name attribute:
The form below has the name attribute:

it was hidden with the jquery .hide() you can inspect the webpage to see it.

Here is is the code:
$("input[name='first_name']").hide();
