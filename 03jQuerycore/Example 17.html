Moving, Copying, and Removing Elements

Moving elements using different approaches.
Make the first list item the last list item:

example:

var li = $( "#myList li:first" ).appendTo( "#myList" );

a un-order list with id = myList and three li elements:

  • item 1
  • item 2
  • item 3


Another approach to the same problem:

$("#myList2").append($("#myList2 li:first"));

a un-order list with id = myList2 and three li elements:

  • item 1
  • item 2
  • item 3

Cloning Elements

Methods such as .appendTo() move the element, but sometimes a copy
of the element is needed instead. In this case, use .clone() first:

$("#myList3 li:first").clone().appendTo("#myList3");

a un-order list with id = myList3 and three li elements:

  • item 1
  • item 2
  • item 3

Creating New Elements

jQuery offers a trivial and elegant way to create new elements using
the same $() method used to make selections:

$( "< p>This is a new paragraph< /p>" );
$( "< li class=\"new\">new list item< /li>" );

// Creating a new element with an attribute object.
$( "< a/>", {
html: "This is a < strong>new< /strong> link",
"class": "new",
href: "foo.html"
});
