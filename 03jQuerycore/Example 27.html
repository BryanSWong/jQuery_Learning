Iterating over jQuery and non-jQuery Objects part 2:

A word of warning: $.map() switches the order of callback arguments.
This was done in order to match the native JavaScript .map() method
made available in ECMAScript 5.

  • item 2
  • item 3
the code:

var arr =[{
id: "a",
tagName:"li"
}, {
id: "b",
tagName: "li"
}, {
id: "c",
tagName: "li"
}];

// Returns [ "a", "b", "c" ]
$("li").map(function(index, element){
return element.id;
}).get();

// Also returns ["a", "b", "c"]
// Note that the value comes first with $.map
$.map(arr, function(value,index){
return value.id;
});

You have to inspect the web page and view console to see results.
