Traversing

Once you've made an initial selection with jQuery, you can
traverse deeper into what was just selected. Traversing can
be broken down into three basic parts: parents, children,
and siblings. jQuery has an abundance of easy-to-use methods
for all these parts.

Here is a outline of html

< div class="grandparent">
< div class="parent">
< div class="child">
< span class="subchild">< /span>
< /div>
< /div>
< div class="surrogateParent1">< /div>
< div class="surrogateParent2">< /div>
< /div>

Parents

The methods for finding the parents from a selection
include .parent(), .parents(), .parentsUntil(), and .closest().

$("span.subchild").parent(); // returns [div.child]

$("span.subchild").parents("div.parent"); // returns [div.parent]

// returns [ div.child, div.parent, div.grandparent ]
$("span.subchild").parents();

// returns [ div.child, div.parent, div.grandparent]
$("span.subchild").parentsUntil("div.grandparent");

$("span.subchild").closest("div"); // returns [div.child]

// returns [ div.child ] as the selector is also included in the search:
$("div.child").closest("div");


you can inspect the web page to see the results in console.

Children

The methods for finding child elements from a selection
include .children() and .find().

console.log($("div.grandparent").children("div"));
// returns [div.parent, div.surrogateParent1, div.surrogateParent2]

console.log($("div.grandparent").find("div"));
// returns [div.child, div.parent, div.surrogateParent1, div.surrogateParent2]

Siblings

The rest of the traversal methods within jQuery all deal with
finding sibling selections. There are a few basic methods as far
as the direction of traversal is concerned. You can find previous
elements with .prev(), next elements with .next(), and both with
.siblings(). There are also a few other methods that build onto
these basic methods: .nextAll(), .nextUntil(), .prevAll() and .prevUntil().

console.log($("div.parent").next()); // returns [ div.surrogateParent1 ]

console.log($("div.parent").prev());
// returns [] as No sibling exists before div.parent

console.log($("div.parent").nextAll());
// returns [ div.surrogateParent1, div.surrogateParent2 ]

console.log($("div.parent").nextAll().first());
// returns [ div.surrogateParent1 ]

console.log($("div.parent").nextAll().last());
// returns [ div.surrogateParent2 ]

console.log($("div.surrogateParent2").prevAll());
// returns [ div.surrogateParent1, div.parent ]

console.log($("div.surrogateParent2").prevAll().first());
// returns [ div.surrogateParent1 ]

console.log($("div.surrogateParent2").prevAll().last());
// returns [ div.parent ]

Use .siblings() to select all siblings:

console.log($("div.parent").siblings());
// returns [ div.surrogateParent1, div.surrogateParent2 ]

console.log($("div.surrogateParent").siblings());
// returns [ div.parent, div.surrogateParent2 ]
