.live() (Deprecated)

All .live() event handlers are bound to the document root by default.

Example:

$("#list li").live("click", function(event){
var elem = $(this);
console.log(elem.text());
});

When we use .live() our event is bound to $( document ).
When the < li > is clicked, bubbling occurs and our click event
is fired for each of the following elements:

< ul >
< div >
< body >
< html >
document root

The last element to receive the click event is document,
this is where our .live() event is bound. .live() will then
check to see if our selector #list li is the element that
triggered the event, if so our event handler is executed.

this example does not work as it really has been deprecated.
